# Negocio Gastronomico Landing Page

Pasos para correr el proyecto:

1. Clonar proyecto
2. ```npm install```
3. Levantar servidor local: ```npm run dev```

Tecnologias utilizadas:

1. React mediante Vite
2. TypeScript
3. React-router-dom
4. Taildwind
